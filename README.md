PreReq:

To be able to run the application the following need to be installed:

-Maven
-AngualrCLI
-Typescript
-Java

Steps for running project:

1. Clone repo
2. Run java code
3. Run Angular
4. Run "localhost:4200"

To run the java code:

 1. mvn clean install
 2. java -jar target/users-0.0.1-SNAPSHOT.jar
 
To run angular:
-npm install 
-npm start
